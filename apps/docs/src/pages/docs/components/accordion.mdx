# Accordion

A vertically stacked set of headings, that are interactive and reveals a section of content.

- [API reference](https://ark-ui.com/docs/solid/components/accordion)

## Basic Usage

<ComponentPreview component={() => {
   const items = ['panel-1', 'panel-2', 'panel-3'];
    return (
      <Accordion>
        <For each={items}>
          {(item) => (
            <AccordionItem value={item}>
              <AccordionTrigger>{item} trigger</AccordionTrigger>
              <AccordionContent>{item} content</AccordionContent>
            </AccordionItem>
          )}
        </For>
      </Accordion>
    );
}}>
