# Alert

An alert displays a short but important message for user's attention.

## Basic Usage

<ComponentPreview component={() => {
  return <Alert><AlertTitle>This is an alert!</AlertTitle></Alert>
}}>
  ```ts
  import { Alert } from '@gothammm/hati';

  export const App = () => (<Alert>
    <AlertTitle>
      This is an alert!
    </AlertTitle>
  </Alert>);
  ```
</ComponentPreview>
