# Installation

Install **Hati** by running either of the commands:

## Usage

**Hati** comes with pre-built css that is required to be imported in your applications entry point.

```ts
import "@gothammm/hati/styles.css";
```

Once that's done, import the components you need to build your applications

<ComponentPreview component={() => <Button>Hey there!</Button>}>
```ts
import { Button } from '@gothammm/hati';

export const App = () => <Button>Hello!</Button>
```
