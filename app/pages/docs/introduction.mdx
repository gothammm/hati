# Introduction

Hati is an open source UI Component Library for SolidJS built with [Ark](https://ark-ui.com "title") and [Tailwind CSS](https://tailwindcss.com/ "title"), it provides a
set of ready to use components to super charge your SolidJS apps.

## Why Hati?

- Batteries Included.
- Bunch of re-usable components, plug and play approach.
- Customizations via simple CSS variables as meant to be.
